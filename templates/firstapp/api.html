C-Mart API's

We provide services to use our data,through our API system,which is integrated with API key structure. In order to access our API's you should have the following things:

1.API Key

2.JWT Token

NOTE :

1. Which ever request you make through the API other than obtaining JWT Token and API Key, it is Mandatory to include API Key and JWT Token in the headers section.

2. the response for all the requests is given in json data ,you need to convert it and use it.

3. In order to perform get request you should a API Key,but to perform post ,update ,delete operations you should be an Admin user.

1.How to obtain the JWT Token ?

- You should become an Authenticated User, In order to be one you should first Sign up into our website.

- URL - 'http://Chaitanyabeemanaboina.pythonanywhere.com/api/token/' ,you should perform a post request to this url.

- In this request you need to add your credentials, namely your username and password.

2.How to obtain the API Key ?

- URL - 'http://Chaitanyabeemanaboina.pythonanywhere.com/api/api-key/', using this url you will get your API Key.

- Including JWT Token into headers section is mandatory,the name for the API Key is set to your username by default.

* While doing All the below requests ,you need to follow all the instructios given in the NOTE section

3.How to get the record ?

- URL - 'http://Chaitanyabeemanaboina.pythonanywhere.com/api/listapi/',perform get request to this url to get the list of items.

4.How to Create a record ?

- URL - 'http://Chaitanyabeemanaboina.pythonanywhere.com/api/createapi/',perform post request to this url for creating an item.

5.How to Update a record ?

- URL - 'http://Chaitanyabeemanaboina.pythonanywhere.com/api/updateapi/1' ,perform update request to this url to update a item.

- here we used 1 in the url as an example ,you can use any id that you want to update ,in place of 1.

6.How to Delete a record ?

- URL - 'http://Chaitanyabeemanaboina.pythonanywhere.com/api/destroyapi/1' ,perform delete request to this url to delete a item.

- here we used 1 in the url as an example ,you can use any id that you want to delete ,in place of 1.
